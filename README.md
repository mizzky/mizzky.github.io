# mizzky.github.io

- Hugo + PaperMod の学習記録用個人ブログです。
- GitHub Pagesでホスティングしています。


## Setup

```bash
git submodule update --init --recursive
```

## Local Server

```bash
mise run server
```

## Build

```bash
mise run build
```

## Create Post

```bash
hugo new posts/post-title-directory/index.md
```
